#PVS Dependencies
Pvs relies on a number of external libraries

[TOC]

#Flask
Flask is a microframework for Python based on Werkzeug and Jinja 2
###Flask-Assets
Flask-Assets helps you to integrate webassets into your Flask application.
###Flask-Login
Flask-Login provides user session management for Flask.
###Flask-SQLAlchemy
Flask-SQLAlchemy is an extension for Flask that adds support for SQLAlchemy to your application.
###Flask-WTF
Flask-WTF offers simple integration with WTForms.
#WTForms
WTForms allows PVS to maintain separation of code and presentation, and keep messy parameters out of the python code.
#SQLAlchemy
SQLAlchemy is the Python SQL toolkit and Object Relational Mapper that gives PVS the full power and flexibility of SQL.
#Less
Less is a CSS pre-processor, meaning that it extends the CSS language, adding features that allow variables, mixins, functions and many other techniques that allows PVS to make CSS that is more maintainable, themable and extendable.
#jQuery
jQuery is a fast, small, and feature-rich JavaScript library.
#Bootstrap
Bootstrap makes front-end web development faster and easier.
#SQLite3
SQLite is a software library that implements a self-contained, serverless, zero-configuration, transactional SQL database engine.
#Markdown
Markdown is a markup language with plain text formatting syntax designed so that it can be converted to HTML
#Pygments
Pygments is a generic syntax highlighter suitable for use in code hosting, forums, wikis or other applications that need to prettify source code.
#Tornado
Tornado is a Python web framework and asynchronous networking library
#Jinja2
Jinja2 is a modern and designer-friendly templating language for Python
#Werkzeug
Werkzeug is a WSGI utility library for Python.
